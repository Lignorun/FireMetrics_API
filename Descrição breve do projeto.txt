Texto explicativo para continuidade do projeto da API de análise ambiental
Resumo do projeto:

Estou desenvolvendo uma API RESTful em Python usando FastAPI que permite a análise temporal e espacial de alertas ambientais na Amazônia (ou outra região definida pelo usuário). Essa API consome dados públicos da plataforma MapBiomas Alerta, que fornece informações sobre:

Alertas de desmatamento mecânico (corte, retirada de vegetação)

Alertas de incêndios florestais (queimadas)

Funcionalidades principais da API:

Receber parâmetros de consulta:

Área geográfica (pode ser por estado, município ou área personalizada)

Intervalo de datas para análise

Tipo de alerta: incêndios ou desmatamento mecânico

Coletar dados da API MapBiomas Alerta:

Buscar os alertas correspondentes ao filtro de área, período e tipo

Os dados incluem área afetada (hectares) e datas dos alertas

Análise estatística:

Agrupar os dados por mês para montar uma série temporal da área afetada

Aplicar uma regressão linear simples para identificar a tendência (crescimento, queda ou estabilidade) da área afetada ao longo do tempo

Geração de resposta visual e textual:

Gerar um gráfico (PNG) que apresenta a série temporal mensal e a linha de tendência

Retornar, junto ao gráfico, um resumo em texto claro, informando se o desmatamento ou incêndios estão aumentando, diminuindo ou estáveis, baseado no coeficiente angular (slope) e no p-valor da regressão

Por que duas situações?

Incêndios: representam a pressão sobre a vegetação por queimadas, possivelmente ligadas a práticas agrícolas (plantação de soja, criação de gado, abertura de áreas)

Desmatamento mecânico: indicam retirada direta da vegetação para usos como madeira ou preparo para outras atividades econômicas

Essa distinção ajuda a interpretar as causas e potenciais usos da terra na região analisada.

Tecnologias usadas:

Python para lógica, cálculos e integração

FastAPI para criar a API rápida e documentada

Requests para consumir a API pública do MapBiomas

Numpy e Scipy para análise estatística (regressão linear)

Matplotlib para gerar gráficos dinâmicos e retornar imagens ao cliente

Objetivo final da API:

Oferecer um serviço simples, mas funcional e visual, que permita a qualquer usuário ou sistema consultar rapidamente a situação ambiental de uma região, com uma indicação clara da tendência de desmatamento e queimadas. Isso pode servir para análise ambiental, tomada de decisão, monitoramento governamental ou suporte para iniciativas privadas.
