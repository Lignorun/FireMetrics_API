About This API

FireMetrics API provides statistical analysis on fire data in Brazil using MapBiomas Fogo datasets. It enables territory searching, grouping retrieval, and access to raw and processed fire data (annual and monthly).

  • GET /data/annual/{local} â€” Get annual fire data for a location
  • GET /data/monthly/{local} â€” Get monthly fire data for a location
  • GET /docs â€” Interactive API documentation (Swagger UI)

Usage Notes

The /data/raw/{local} endpoint fetches and caches the raw fire data for the specified location. Subsequent requests to /data/annual/{local} or /data/monthly/{local} serve data from the cache to improve performance.
