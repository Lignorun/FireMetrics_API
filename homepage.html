About the API

The FireMetrics API was developed to provide statistical analysis of fire data in Brazil, using the historical data series from MapBiomas Fogo. Our API allows searching for territories, getting grouping types, and accessing processed burned area data, both annual and monthly.

How to Use

To get burned area data, the ideal process is:

  1. Use the /territories/{search_term} endpoint to find the local_id of the desired territory.
  2. Make the first call to the /data/raw/{local_id} endpoint. This will fetch the data from MapBiomas and cache it.
  3. Then, call the /data/annual/{local_id} or /data/monthly/{local_id} endpoints to access the processed data instantly, using the cache.

This caching approach ensures that after the first request, subsequent analyses are much faster.
